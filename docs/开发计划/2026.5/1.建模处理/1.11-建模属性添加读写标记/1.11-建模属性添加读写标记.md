# 1.11 建模属性添加读写标记

## 任务信息

- **任务号**: 1.11
- **任务名称**: 建模属性添加读写标记
- **责任人**: 郑琳
- **任务类型**: 代码修改
- **任务描述**: 在 `IModelProperty` 中添加 `bool CanRead` 和 `bool CanWrite` 属性

## 功能概述

为建模属性接口 `IModelProperty` 添加读写标记属性，以标识属性是否可读取或可写入。这使得调用方能够通过建模属性对象快速判断属性的可访问性，而无需依赖反射或其他运行时检查。

**重要约束**：
- 接口定义必须兼容 .NET Standard 2.0，不能使用 `object?` 等可空类型
- 所有公共成员必须有 XML 文档注释
- 源生成器生成的属性实现类需要实现这两个属性

## 技术背景

### 现有实现

`IModelProperty` 接口目前包含以下成员：

```csharp
namespace Delly.Modeling;

/// <summary>
/// 建模属性接口
/// </summary>
public interface IModelProperty
{
    /// <summary>
    /// 获取属性名称
    /// </summary>
    string Name { get; }

    /// <summary>
    /// 获取属性类型
    /// </summary>
    Type PropertyType { get; }

    /// <summary>
    /// 获取属性值
    /// </summary>
    /// <param name="model">模型实例</param>
    /// <param name="instance">对象实例</param>
    /// <returns>属性值</returns>
    object GetValue(Model model, object instance);

    /// <summary>
    /// 设置属性值
    /// </summary>
    /// <param name="model">模型实例</param>
    /// <param name="instance">对象实例</param>
    /// <param name="value">属性值</param>
    void SetValue(Model model, object instance, object value);
}
```

### 新增需求

在 `IModelProperty` 接口中添加两个属性：

1. **CanRead**: 标识属性是否可读取
2. **CanWrite**: 标识属性是否可写入

对于标准的 C# 属性：
- 只读属性（只有 `get`）：`CanRead = true`, `CanWrite = false`
- 只写属性（只有 `set`）：`CanRead = false`, `CanWrite = true`
- 读写属性（同时有 `get` 和 `set`）：`CanRead = true`, `CanWrite = true`

## 实现方案

### 1. 修改 IModelProperty 接口

在 `Delly.Modeling/IModelProperty.cs` 中添加两个新属性：

```csharp
namespace Delly.Modeling;

/// <summary>
/// 建模属性接口

public interface IModelProperty
```

### 2. 修改源生成器

在 `ModelableSourceGenerator.cs` 中：

1. 在生成属性类时添加 `CanRead` 和 `CanWrite` 属性
2. 根据目标属性的访问修饰符（`get`/`set`）确定属性值

```csharp
private static string GeneratePropertyClassTemplate(
    string className,
    string propertyName,
    string typeName,
    bool hasGetter,
    bool hasSetter)
{
    return $$"""
    #if !NETSTANDARD2_0
    #nullable enable
    #endif

    /// <summary>
    /// {{className}} 的 {{propertyName}} 属性模型
    /// </summary>
    public sealed class mp__{{className}}_{{propertyName}} : IModelProperty
    {
        /// <summary>
        /// 获取属性名称
        /// </summary>
        public string Name => "{{propertyName}}";

        /// <summary>
        /// 获取属性类型
        /// </summary>
        public Type PropertyType => typeof({{typeName}});

        /// <summary>
        /// 获取属性是否可读取
        /// </summary>
        public bool CanRead => {{(hasGetter ? "true" : "false")}};

        /// <summary>
        /// 获取属性是否可写入
        /// </summary>
        public bool CanWrite => {{(hasSetter ? "true" : "false")}};

        /// <summary>
        /// 获取属性值
        /// </summary>
        /// <param name="model">模型实例</param>
        /// <param name="instance">对象实例</param>
        /// <returns>属性值</returns>
    #if !NETSTANDARD2_0
        public object? GetValue(Model model, object instance)
    #else
        public object GetValue(Model model, object instance)
    #endif
        {
            var property = model.GetProperty(model, instance, "{{propertyName}}") as mp__{{className}}_{{propertyName}};
            if (property == null)
            {
                return default;
            }
            return property.GetValue(model, instance);
        }

        /// <summary>
        /// 设置属性值
        /// </summary>
        /// <param name="model">模型实例</param>
        /// <param name="instance">对象实例</param>
        /// <param name="value">属性值</param>
    #if !NETSTANDARD2_0
        public void SetValue(Model model, object instance, object? value)
    #else
        public void SetValue(Model model, object instance, object value)
    #endif
        {
            var property = model.GetProperty(model, instance, "{{propertyName}}") as mp__{{className}}_{{propertyName}};
            if (property == null)
            {
                return;
            }
            property.SetValue(model, instance, value);
        }
    }
    """;
}
```

### 3. 修改属性信息收集逻辑

在 `ModelableSourceGenerator.cs` 的 `CollectProperties` 方法中，需要收集属性的可访问性信息：

```csharp
private static List<PropertyInfo> CollectProperties(IPropertySymbol propertySymbol)
{
    var properties = new List<PropertyInfo>();

    // 收集属性的可访问性信息
    var hasGetter = propertySymbol.GetMethod != null;
    var hasSetter = propertySymbol.SetMethod != null;

    // 将信息传递给模板生成函数
    properties.Add(new PropertyInfo
    {
        Name = propertySymbol.Name,
        TypeName = propertySymbol.Type.ToDisplayString(),
        HasGetter = hasGetter,
        HasSetter = hasSetter
    });

    return properties;
}
```

## 使用示例

### 查询属性可访问性

```csharp
[Modelable]
public partial class User
{
    public string Id { get; } = string.Empty;           // 只读: CanRead=true, CanWrite=false
    public string Name { get; set; } = string.Empty;    // 读写: CanRead=true, CanWrite=true
    private string _secret { get; set; }                 // 不生成建模属性
}

// 使用示例
var model = Model<User>.Instance;
var properties = model.GetProperties();

foreach (var propertyName in properties)
{
    var property = model.GetProperty(User.GetEntityModel(), new User(), propertyName);
    Console.WriteLine($"{property.Name}: Read={property.CanRead}, Write={property.CanWrite}");
}

// 输出:
// Id: Read=True, Write=False
// Name: Read=True, Write=True
```

### 条件读写操作

```csharp
public static void SetPropertySafe(Model model, object instance, string propertyName, object value)
{
    var property = model.GetProperty(model, instance, propertyName);
    if (property is IModelProperty modelProperty)
    {
        if (!modelProperty.CanWrite)
        {
            throw new InvalidOperationException($"属性 '{propertyName}' 是只读的");
        }
        modelProperty.SetValue(model, instance, value);
    }
}
```
