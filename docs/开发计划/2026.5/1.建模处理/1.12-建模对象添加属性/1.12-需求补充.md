# 需求补充

## 需求澄清

### IsValue 属性的语义

`IsValue` 属性用于区分基础值类型和一般对象类型。基础值类型包括：
- .NET 基础值类型（int, long, bool, double, decimal 等）
- string 类型（特殊处理，视为值类型）
- DateTime 和 Guid 等常用值类型

### 值类型判断规则

| 类型 | IsValue |
|:-----|:--------:|
| string | true |
| int / Int32 | true |
| long / Int64 | true |
| bool / Boolean | true |
| double / Double | true |
| decimal / Decimal | true |
| DateTime | true |
| Guid | true |
| enum | true |
| struct | true |
| Nullable<T> | true |
| 泛型类型（如 List<T>） | false |
| 其他 class 类型 | false |
