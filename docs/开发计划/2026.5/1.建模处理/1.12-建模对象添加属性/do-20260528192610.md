## Prompt 文档理解

### 功能概述
在 `IBaseModel` 接口中新增 `IsValue` 属性，用于区分基础值对象和一般对象类型。

### IsValue 属性语义
- 基础值类型（int, long, bool, double, decimal 等）返回 `true`
- string 类型返回 `true`
- DateTime 和 Guid 等常用值类型返回 `true`
- enum, struct, Nullable<T> 返回 `true`
- 其他 class 类型返回 `false`

### 实现要求
1. 修改 `IBaseModel.cs` 接口添加 `IsValue` 属性定义
2. 修改所有 Models 目录下的 Model 类实现 `IsValue` 属性（返回 `true`）
3. 修改源生成器为生成的类添加 `IsValue` 属性（返回 `false`）
4. 在 Deme 项目中添加示例代码验证功能
