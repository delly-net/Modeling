# 需求补充

1. **创建新的的特性**
   - 名称：MoQueryAttribute
   - 为 MoQueryAttribute 特性添加源生成支持
   - MoQueryAttribute 不支持与 MoColumnAttribute 搭配工作

2. **兼容现有实体建模**
   - 针对 MoQueryAttribute 的源生成代码创建 IEntityModel
   - MoTableAttribute 与 MoQueryAttribute 都是为 Sql查询 提供 IEntityModel 对象支持
   - MoTableAttribute 与 MoQueryAttribute 互斥，不允许同时指定，需要添加逻辑控制

3. **源生成**
   - MoTableAttribute 与 MoQueryAttribute 共享一个源生成器，且生成的类命名相同
   - 升级现有源生成器实现需求
   - MoQueryAttribute 支持所有公开属性，无需指定特殊特性
