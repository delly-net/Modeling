### 新增规范细节

**if 语句强制处理规则**：
1. if 语句必须使用大括号
2. 判断 if 语句所属代码段中是否只有一行语句
   - 是 → 判断 if 语句 + 代码段是否超过 120 字符
     - **超过 120 字符** → 代码强制折行
     - **未超过 120 字符** → 代码强制压缩为单行

### 示例对比

```csharp
// 超过 120 字符 → 折行
if (条件)
{
    代码行;
}

// 未超过 120 字符 → 压缩为单行
if (条件) { 代码行; }
```
