# 1.2 建模工厂支持

## 任务信息

- **任务号**: 1.2
- **任务名称**: 建模工厂支持
- **责任人**: 郑琳
- **任务类型**: 功能开发
- **任务描述**: 添加针对建模同一管理的建模工厂能力支持

## 功能概述

添加建模工厂能力，用于统一管理项目中所有的实体模型对象。通过模型集合和工厂的组合，提供灵活的模型注册、获取、遍历等功能，支持动态注册模式，兼容模块化开发。

**重要约束**：
- 工厂提供统一的实体模型访问入口
- 采用手动注册方式，支持模块化开发
- 完全支持 AOT 编译，不使用反射和 Type 类型
- 只支持 `IEntityModel`，不支持 `IModel`
- 模型唯一性使用类全称 `{Namespace}.{ClassName}` 作为标识
- 静态工具定义使用 `Utils` 结尾
- MoSetAttribute 源生成器独立实现，不修改现有源生成器

## 技术背景

### 现有实现

当前项目中，每个标记了建模特性的类通过源代码生成器生成独立的模型获取方法：

| 特性 | 生成方法 | 返回类型 | 命名空间 |
|:----|:--------|:---------|:---------|
| `MoTable` | `GetEntityModel()` | `IEntityModel` | 类所在命名空间 |
| `MoQuery` | `GetEntityModel()` | `IEntityModel` | 类所在命名空间 |

这些方法分散在各个类中，没有统一的访问入口，也无法在运行时获取所有已注册的模型列表。

### 新增需求

**建模工厂体系**：
- 提供 `IEntityModelSet` 接口进行建模信息的自动收集
- 提供 `MoSetAttribute` 特性进行当前程序集中建模信息的自动收集
- 提供 `IEntityModelFactory` 接口定义来管理所有的实体建模
- 提供 `DefaultEntityModelFactory` 标准实现
- 支持动态注册（运行时）
- 使用类全称 `{Namespace}.{ClassName}` 作为模型的唯一标识
- 兼容模块化开发，需要手动实例化工厂并填充建模信息
- MoSetAttribute 源生成器独立实现，不修改现有源生成器

### 3. 新增 IEntityModelFactory 接口

```csharp
namespace Delly.Modeling;

/// <summary>
/// 实体建模工厂接口，定义实体建模管理的标准
/// </summary>
public interface IEntityModelFactory
{
    /// <summary>
    /// 获取指定类全称的实体模型
    /// </summary>
    /// <param name="fullName">类全称，格式为 {Namespace}.{ClassName}</param>
    /// <returns>实体模型实例，如不存在则返回 null</returns>
    IEntityModel? GetModel(string fullName);

    /// <summary>
    /// 获取指定表名的实体模型
    /// </summary>
    /// <param name="tableName">表名/查询对象名称</param>
    /// <returns>实体模型实例，如不存在则返回 null</returns>
    IEntityModel? GetModelByTableName(string tableName);

    /// <summary>
    /// 获取所有实体模型
    /// </summary>
    /// <returns>所有已注册的实体模型</returns>
    IEnumerable<IEntityModel> GetAllModels();

    /// <summary>
    /// 获取指定命名空间的所有实体模型
    /// </summary>
    /// <param name="ns">命名空间</param>
    /// <returns>指定命名空间的所有实体模型</returns>
    IEnumerable<IEntityModel> GetModelsByNamespace(string ns);

    /// <summary>
    /// 检查是否存在指定类全称的实体模型
    /// </summary>
    /// <param name="fullName">类全称</param>
    /// <returns>是否存在</returns>
    bool HasModel(string fullName);

    /// <summary>
    /// 获取模型总数
    /// </summary>
    int Count { get; }

    /// <summary>
    /// 添加建模集合
    /// </summary>
    /// <param name="modelSet">建模集合</param>
    void AddSet(IEntityModelSet modelSet);

    /// <summary>
    /// 添加单个实体模型
    /// </summary>
    /// <param name="model">实体模型</param>
    void Add(IEntityModel model);
}
```

### 4. 新增 DefaultEntityModelFactory 实现

```csharp
namespace Delly.Modeling;

/// <summary>
/// 默认实体建模工厂实现
/// </summary>
public sealed class DefaultEntityModelFactory : IEntityModelFactory
{
    private readonly Dictionary<string, IEntityModel> _models;

    /// <summary>
    /// 初始化默认实体建模工厂
    /// </summary>
    public DefaultEntityModelFactory()
    {
        _models = [];
    }

    /// <summary>
    /// 获取指定类全称的实体模型
    /// </summary>
    public IEntityModel? GetModel(string fullName)
    {
        return _models.TryGetValue(fullName, out var model) ? model : null;
    }

    /// <summary>
    /// 获取指定表名的实体模型
    /// </summary>
    public IEntityModel? GetModelByTableName(string tableName)
    {
        return _models.Values.FirstOrDefault(m => m.Name == tableName);
    }

    /// <summary>
    /// 获取所有实体模型
    /// </summary>
    public IEnumerable<IEntityModel> GetAllModels()
    {
        return _models.Values;
    }

    /// <summary>
    /// 获取指定命名空间的所有实体模型
    /// </summary>
    public IEnumerable<IEntityModel> GetModelsByNamespace(string ns)
    {
        return _models.Values.Where(m => m.Namespace == ns);
    }

    /// <summary>
    /// 检查是否存在指定类全称的实体模型
    /// </summary>
    public bool HasModel(string fullName)
    {
        return _models.ContainsKey(fullName);
    }

    /// <summary>
    /// 获取模型总数
    /// </summary>
    public int Count => _models.Count;

    /// <summary>
    /// 添加建模集合
    /// </summary>
    public void AddSet(IEntityModelSet modelSet)
    {
        var models = modelSet.GetModels();
        foreach (var model in models)
        {
            var fullName = $"{model.Namespace}.{model.ClassName}";
            _models[fullName] = model;
        }
    }

    /// <summary>
    /// 添加单个实体模型
    /// </summary>
    public void Add(IEntityModel model)
    {
        var fullName = $"{model.Namespace}.{model.ClassName}";
        _models[fullName] = model;
    }
}
```

## 使用示例

### 基本使用方式

```csharp
// 定义建模集合
[MoSet]
public partial class EntityModelSet;

// 定义模型类
[MoTable("orders")]
public partial class Order
{
    [MoColumn(Name = "id", Key = true, Type = ColumnType.LONG)]
    public long Id { get; set; }
}

[MoQuery]
public partial class OrderQuery
{
    public DateTime? FromDate { get; set; }
    public DateTime? ToDate { get; set; }
}

// 手动创建工厂并注册模型
var factory = new DefaultEntityModelFactory();
var entitySet = new EntityModelSet();

// 通过建模集合添加
factory.AddSet(entitySet);

// 或者直接添加单个模型
factory.Add(Order.GetEntityModel());

// 使用工厂（通过类全称）
var orderModel = factory.GetModel("MyApp.Models.Order");
if (orderModel != null)
{
    Console.WriteLine($"表名: {orderModel.Name}");
    Console.WriteLine($"类名: {orderModel.ClassName}");
}

// 使用工厂（通过表名）
var orderByTableName = factory.GetModelByTableName("orders");

// 获取所有模型
var allModels = factory.GetAllModels();
Console.WriteLine($"总模型数: {factory.Count}");

// 获取指定命名空间的模型
var appModels = factory.GetModelsByNamespace("MyApp.Models");
foreach (var model in appModels)
{
    Console.WriteLine($"模型: {model.ClassName}");
}
```

### 模块化开发方式

```csharp
// 模块 A
public class ModuleA
{
    private readonly DefaultEntityModelFactory _factory;

    public ModuleA(DefaultEntityModelFactory factory)
    {
        _factory = factory;
    }

    public void RegisterModels()
    {
        _factory.Add(Order.GetEntityModel());
        _factory.Add(User.GetEntityModel());
    }
}

// 模块 B
public class ModuleB
{
    private readonly DefaultEntityModelFactory _factory;

    public ModuleB(DefaultEntityModelFactory factory)
    {
        _factory = factory;
    }

    public void RegisterModels()
    {
        _factory.Add(Product.GetEntityModel());
        _factory.Add(Category.GetEntityModel());
    }
}

// 主程序
var factory = new DefaultEntityModelFactory();
var moduleA = new ModuleA(factory);
var moduleB = new ModuleB(factory);

moduleA.RegisterModels();
moduleB.RegisterModels();

// 使用工厂
var allModels = factory.GetAllModels();
```

### 自定义建模集合

```csharp
// 创建自定义建模集合
public class CustomModelSet : IEntityModelSet
{
    private readonly List<IEntityModel> _models = [];

    public void Add(IEntityModel model) => _models.Add(model);

    public IReadOnlyList<IEntityModel> GetModels() => _models.ToArray();

    public int Count => _models.Count;
}

// 使用自定义集合
var factory = new DefaultEntityModelFactory();
var customSet = new CustomModelSet();

customSet.Add(Order.GetEntityModel());
customSet.Add(OrderQuery.GetEntityModel());

factory.AddSet(customSet);
```
