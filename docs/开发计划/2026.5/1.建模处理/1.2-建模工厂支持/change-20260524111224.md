# 变更记录

## 原文档内容

原文档包含完整的建模工厂体系设计，包括 `IEntityModelSet`、`MoSetAttribute`、`IEntityModelFactory`、`DefaultEntityModelFactory` 和 `ModelFactory` 的定义和实现。

## 变更内容

1. **修改 IEntityModelSet.GetModels() 返回类型**：从 `IEntityModel[]` 改为 `IReadOnlyList<IEntityModel>`，符合不可变集合规范（有数量概念）
2. **确认 AOT 编译规范**：文档中已明确完全支持 AOT 编译，不使用反射和 Type 类型

## 变更原因

根据 `1.2-需求补充.md` 中的开发规范补充：
- 不可变集合规范：有数量概念 → 使用 IReadOnlyList，无数量概念 → 使用 IEnumerable
- `IEntityModelSet.GetModels()` 有 `Count` 属性，属于有数量概念的集合

## 可能产生的影响

1. API 更加符合不可变集合的最佳实践
2. 与现有代码兼容，`IEntityModel[]` 实现了 `IReadOnlyList<IEntityModel>` 接口
3. 无破坏性变更
