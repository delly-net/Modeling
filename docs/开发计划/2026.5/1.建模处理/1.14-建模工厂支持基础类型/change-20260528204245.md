## 变更内容

### 1. 泛型约束移除

**变更原因**: 需求补充文档新增"需求完善"章节

**变更内容**:
- 原方案：`GetModel<T>()` 方法使用 `where T : class` 约束
- 新方案：去掉 `where T : class` 约束，支持所有类型（包括值类型）

**技术方案调整**:
- `GetModel<T>()` 方法移除 `where T : class` 约束
- `TryGetModelFromModelSets<T>()` 辅助方法移除 `where T : class` 约束
- 方法可以直接支持基础类型泛型调用（如 `GetModel<int>()`、`GetModel<bool>()` 等）

### 2. 代码示例更新

**使用示例**调整：
```csharp
// 去掉约束后，可以直接对值类型使用泛型方法
var intModel = factory.GetModel<int>(); // 现在可以直接调用
var boolModel = factory.GetModel<bool>();
```

## 可能产生的影响

1. **API 变更**: `GetModel<T>()` 方法签名改变，移除泛型约束
2. **调用简化**: 不再需要通过字符串名称查询基础类型，可直接使用泛型
3. **类型安全**: 编译时即可检查基础类型支持情况
4. **辅助方法**: `TryGetModelFromModelSets<T>()` 也需要移除约束
