## 功能概述

在 `IBaseModel` 接口中添加 `Parse` 和 `TryParse` 函数，提供对象值解析能力。`Parse` 函数用于将输入对象解析为目标类型实例，`TryParse` 函数提供安全的解析方式，解析失败时返回 `null`。

**重要约束**：
- `Parse` 函数解析失败时应抛出异常
- `TryParse` 函数解析失败时应返回 `null`，不应抛出异常
- 解析逻辑需要支持常见类型转换
- TODO: 确认是否需要支持自定义解析器注册

## 技术背景

### 现有实现

当前 `IBaseModel` 接口定义如下：

```csharp
namespace Delly.Modeling
{
    /// <summary>
    /// 模型对象接口，提供属性访问能力
    /// </summary>
    public interface IBaseModel
    {
        /// <summary>
        /// 名称
        /// </summary>
        string Name { get; }

        /// <summary>
        /// 类型全名称
        /// </summary>
        string Namespace { get; }

        /// <summary>
        /// 创建模型类型的新实例
        /// </summary>
        /// <param name="args">构造函数参数数组</param>
        /// <returns>模型类型的新实例</returns>
        object CreateInstance(params object[] args);

        /// <summary>
        /// 模型类型信息，源生成阶段使用 typeof(T) 赋值
        /// </summary>
        Type ClassType { get; }
    }
}
```

### 新增需求

需要在 `IBaseModel` 接口中添加以下两个函数：

1. `Parse` - 将输入对象解析为目标类型实例，解析失败时抛出异常
2. `TryParse` - 将输入对象解析为目标类型实例，解析失败时返回 `null`

## 实现方案

### 1. 修改 IBaseModel 接口

在 `IBaseModel` 接口中添加以下函数：

```csharp
/// <summary>
/// 将输入对象解析为目标类型实例
/// </summary>
/// <param name="obj">输入对象</param>
/// <returns>目标类型实例</returns>
/// <exception cref="ArgumentException">当对象无法转换为目标类型时抛出</exception>
object Parse(object? obj);

/// <summary>
/// 尝试将输入对象解析为目标类型实例
/// </summary>
/// <param name="obj">输入对象</param>
/// <returns>目标类型实例，解析失败时返回 null</returns>
object? TryParse(object? obj);
```

### 2. 源生成器实现

需要在以下源生成器中添加 `Parse` 和 `TryParse` 函数的生成代码：

| 源生成器 | 特性 | 生成的类 |
|:--------|:----|:--------|
| `ModelableSourceGenerator` | `ModelableAttribute` | `emc__{ClassName}` |
| `ModelTableSourceGenerator` | `MoTableAttribute` | `emc__{ClassName}` |
| `ModelTableSourceGenerator` | `MoQueryAttribute` | `emc__{ClassName}` |

生成的 `Parse` 和 `TryParse` 函数示例：

```csharp
// emc__{ClassName} 类中
public override object Parse(object? obj)
{
    if (obj == null)
        throw new ArgumentException("Cannot parse null value");

    if (obj is {ClassName} target)
        return target;

    if (obj is string str)
        return ParseFromString(str);  // TODO: 确认是否需要字符串解析支持

    throw new ArgumentException($"Cannot convert {obj.GetType().Name} to {ClassName}");
}

public override object? TryParse(object? obj)
{
    if (obj == null)
        return null;

    try
    {
        return Parse(obj);
    }
    catch
    {
        return null;
    }
}
```
