## 变更内容

### 1. ParsableAttribute 特性定义变更

**原始实现**：
```csharp
public sealed class ParsableAttribute : Attribute
{
    public ParsableAttribute(Type targetType)
    {
        TargetType = targetType;
    }
    public Type TargetType { get; }
}
```

**新实现**（按照需求补充）：
```csharp
[AttributeUsage(AttributeTargets.Class, AllowMultiple = false, Inherited = false)]
public sealed class ParsableAttribute : Attribute
{
}
```

### 2. 目标类型推断方式变更

**原始方式**：
- 用户使用 `[Parsable(typeof(Order))]` 特性
- 源生成器从特性参数获取目标类型

**新方式**：
- 用户使用 `[Parsable]` 特性
- 源生成器从 `IParsable<T>` 的泛型类型参数推断目标类型

### 3. 用户使用方式变更

**原始方式**：
```csharp
[Parsable(typeof(Order))]
public class OrderParser : IParsable<Order>
```

**新方式**：
```csharp
[Parsable]
public class OrderParser : IParsable<Order>
```

## 变更原因

需求补充要求 `ParsableAttribute` 特性无需添加参数，根据 `IParsable<T>` 的 T 实现类来定义目标。这样可以减少用户的代码复杂度。

## 可能产生的影响

1. **用户代码简化**：
   - 无需使用 `typeof()` 表达式
   - 避免拼写错误

2. **源生成器复杂度增加**：
   - 需要解析 `IParsable<T>` 的泛型类型参数
   - 需要验证 `IParsable<T>` 的 T 与目标类型匹配
