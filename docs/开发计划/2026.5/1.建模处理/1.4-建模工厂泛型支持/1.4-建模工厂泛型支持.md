# 1.4 建模工厂泛型支持

## 任务信息
- **任务号**: 1.4
- **任务名称**: 建模工厂泛型支持
- **责任人**: 郑琳
- **任务类型**: 功能开发
- **任务描述**: 在建模工厂中添加根据泛型获取建模能力

## 功能概述

在建模工厂（`IEntityModelFactory` / `DefaultEntityModelFactory`）中添加泛型方法支持，允许通过类型参数直接获取对应类型的实体模型。工厂通过委托已注册的 `IEntityModelSet` 集合的泛型方法来实现类型安全的 API。

**重要约束**：
- 必须保持与现有接口的向后兼容性
- 泛型实现委托给 `IEntityModelSet` 的泛型方法实现
- 只允许通过 `AddSet` 方法注册集合，`Add` 方法不影响 `GetModel<T>` 结果
- 同名的 `IEntityModelSet` 只允许注册一次，后续调用将被忽略
- 泛型方法仅提供以下两个：`GetModel<T>()` 和 `TryGetModel<T>()`

## 技术背景

### 现有实现

**现有工厂接口**：
```csharp
/// <summary>
/// 实体建模工厂接口，定义实体建模管理的标准
/// </summary>
public interface IEntityModelFactory
{
    /// <summary>
    /// 获取指定类全称的实体模型
    /// </summary>
    /// <param name="fullName">类全称，格式为 {Namespace}.{ClassName}</param>
    /// <returns>实体模型实例，如不存在则返回 null</returns>
    IEntityModel? GetModel(string fullName);

    /// <summary>
    /// 获取指定表名的实体模型
    /// </summary>
    /// <param name="tableName">表名/查询对象名称</param>
    /// <returns>实体模型实例，如不存在则返回 null</returns>
    IEntityModel? GetModelByTableName(string tableName);

    /// <summary>
    /// 获取所有实体模型
    /// </summary>
    /// <returns>所有已注册的实体模型</returns>
    IEnumerable<IEntityModel> GetAllModels();

    /// <summary>
    /// 获取指定命名空间的所有实体模型
    /// </summary>
    /// <param name="ns">命名空间</param>
    /// <returns>指定命名空间的所有实体模型</returns>
    IEnumerable<IEntityModel> GetModelsByNamespace(string ns);

    /// <summary>
    /// 检查是否存在指定类全称的实体模型
    /// </summary>
    /// <param name="fullName">类全称</param>
    /// <returns>是否存在</returns>
    bool HasModel(string fullName);

    /// <summary>
    /// 获取模型总数
    /// </summary>
    int Count { get; }

    /// <summary>
    /// 添加建模集合
    /// </summary>
    /// <param name="modelSet">建模集合</param>
    void AddSet(IEntityModelSet modelSet);

    /// <summary>
    /// 添加单个实体模型
    /// </summary>
    /// <param name="model">实体模型</param>
    void Add(IEntityModel model);
}
```

**现有工厂实现**：
```csharp
/// <summary>
/// 默认实体建模工厂实现
/// </summary>
public sealed class DefaultEntityModelFactory : IEntityModelFactory
{
    private readonly Dictionary<string, IEntityModel> _models;

    public DefaultEntityModelFactory()
    {
        _models = new Dictionary<string, IEntityModel>();
    }

    public IEntityModel? GetModel(string fullName)
    {
        return _models.TryGetValue(fullName, out var model) ? model : null;
    }

    public IEntityModel? GetModelByTableName(string tableName)
    {
        return _models.Values.FirstOrDefault(m => m.Name == tableName);
    }

    public IEnumerable<IEntityModel> GetAllModels()
    {
        return _models.Values;
    }

    public IEnumerable<IEntityModel> GetModelsByNamespace(string ns)
    {
        return _models.Values.Where(m => m.Namespace == ns);
    }

    public bool HasModel(string fullName)
    {
        return _models.ContainsKey(fullName);
    }

    public int Count => _models.Count;

    public void AddSet(IEntityModelSet modelSet)
    {
        var models = modelSet.GetModels();
        foreach (var model in models)
        {
            var fullName = $"{model.Namespace}.{model.ClassName}";
            _models[fullName] = model;
        }
    }

    public void Add(IEntityModel model)
    {
        var fullName = $"{model.Namespace}.{model.ClassName}";
        _models[fullName] = model;
    }
}
```

**现有集合接口**（任务 1.3 已实现泛型方法）：
```csharp
/// <summary>
/// 实体模型集合接口，用于建模信息的自动收集
/// </summary>
public interface IEntityModelSet
{
    /// <summary>
    /// 获取集合中的所有实体模型
    /// </summary>
    IReadOnlyList<IEntityModel> GetModels();

    /// <summary>
    /// 获取集合中的模型数量
    /// </summary>
    int Count { get; }

    /// <summary>
    /// 获取指定类型的模型
    /// </summary>
    /// <typeparam name="T">模型类型</typeparam>
    /// <returns>指定类型的模型</returns>
    IEntityModel GetModel<T>() where T : class;

    /// <summary>
    /// 尝试获取指定类型的模型
    /// </summary>
    /// <typeparam name="T">模型类型</typeparam>
    /// <returns>模型对象，未找到时返回 null</returns>
    IEntityModel? TryGetModel<T>() where T : class;
}
```

**现有使用方式**：
```csharp
var factory = new DefaultEntityModelFactory();
factory.Add(User.GetEntityModel());
factory.Add(Order.GetEntityModel());

// 通过类全称获取

// 通过表名获取

```

### 新增需求

为建模工厂添加泛型方法，通过委托已注册的 `IEntityModelSet` 的泛型方法实现：

```csharp
/// <summary>
/// 获取指定类型的实体模型
/// </summary>
/// <typeparam name="T">实体类型</typeparam>
/// <returns>指定类型的实体模型</returns>
/// <exception cref="NotSupportedException">当类型未在任何已注册集合中找到时抛出</exception>
IEntityModel GetModel<T>() where T : class;

/// <summary>
/// 尝试获取指定类型的实体模型
/// </summary>
/// <typeparam name="T">实体类型</typeparam>
/// <returns>指定类型的实体模型，未找到时返回 null</returns>
IEntityModel? TryGetModel<T>() where T : class;
```

**实现方式**：
- 维护全局 `IEntityModelSet` 列表
- 调用 `AddSet(IEntityModelSet modelSet)` 时将 modelSet 保存到列表
- 调用 `GetModel<T>()` 时，遍历 `IEntityModelSet` 列表，依次调用每个 set 的 `TryGetModel<T>()` 方法
- `Add(IEntityModel model)` 方法不影响 `GetModel<T>` 的结果
- 同名的 `IEntityModelSet` 只允许注册一次，后续调用将被忽略

### 2. 关键实现细节

**同名 set 检测**：
```csharp
private readonly HashSet<IEntityModelSet> _registeredSets = [];

public void AddSet(IEntityModelSet modelSet)
{
    if (_registeredSets.Add(modelSet))
    {
        // 首次注册，添加到列表
        _modelSets.Add(modelSet);
    }
    // 重复调用时跳过
}
```

**泛型方法实现**：
```csharp
public IEntityModel GetModel<T>() where T : class
{
    foreach (var set in _modelSets)
    {
        var model = set.TryGetModel<T>();
        if (model != null)
        {
            return model;
        }
    }
    throw new NotSupportedException($"类型 {typeof(T).FullName} 未在已注册的建模集合中找到");
}

public IEntityModel? TryGetModel<T>() where T : class
{
    foreach (var set in _modelSets)
    {
        var model = set.TryGetModel<T>();
        if (model != null)
        {
            return model;
        }
    }
    return null;
}
```

## 使用示例

### 示例 1：基本使用

```csharp
// 定义建模集合
[MoSet]
public partial class EntityModelSet;

// 定义模型类
[MoTable("users")]
public partial class User
{
    public long Id { get; set; }
    public string Name { get; set; }
}

[MoTable("orders")]
public partial class Order
{
    public long Id { get; set; }
    public long UserId { get; set; }
}

// 使用工厂
var factory = new DefaultEntityModelFactory();
var entitySet = new EntityModelSet();

// 注册集合（必需，否则泛型方法无法获取）
factory.AddSet(entitySet);

// 通过泛型获取模型
var userModel = factory.GetModel<User>();
Console.WriteLine($"表名: {userModel.Name}");

// 使用 TryGetModel 避免异常
var productModel = factory.TryGetModel<Product>();
if (productModel != null)
{
    Console.WriteLine($"表名: {productModel.Name}");
}
```

### 示例 2：多个集合

```csharp
// 定义多个集合
[MoSet]
public partial class ModuleASet;

[MoSet]
public partial class ModuleBSet;

// 使用工厂
var factory = new DefaultEntityModelFactory();

// 注册多个集合
factory.AddSet(new ModuleASet());
factory.AddSet(new ModuleBSet());

// 泛型方法会遍历所有已注册集合
var user = factory.GetModel<User>();
var order = factory.GetModel<Order>();
```

### 示例 3：Add 方法不影响泛型获取

```csharp
var factory = new DefaultEntityModelFactory();
var entitySet = new EntityModelSet();

// 直接 Add 注册
factory.Add(User.GetEntityModel());

// 泛型获取失败，因为没有注册集合
var user = factory.TryGetModel<User>(); // 返回 null

// 注册集合后才能使用泛型
factory.AddSet(entitySet);
user = factory.GetModel<User>(); // 成功获取
```

## 验收标准

- `GetModel<T>()` 能正确返回指定类型的模型
- `GetModel<T>()` 对未注册类型抛出 `NotSupportedException`
- `TryGetModel<T>()` 能正确返回模型对象或 null
- 泛型方法只在已通过 `AddSet` 注册的集合中查找
- 单独使用 `Add` 方法注册的模型无法通过泛型方法获取
- 同名的 `IEntityModelSet` 只被注册一次
- 单元测试覆盖率 ≥ 80%
- 文档更新完成，示例代码可运行
- 通过 AOT 编译测试
- 兼容 .NET Standard 2.0 和 .NET 10.0，nullable 注解正确处理
