# 1.3 建模集泛型支持

## 任务信息
- **任务号**: 1.3
- **任务名称**: 建模集泛型支持
- **责任人**: 郑琳
- **任务类型**: 功能开发
- **任务描述**: 在建模集源生成中添加泛型转换支持

## 功能概述

建模集（`MoSetAttribute`）目前只支持生成实现 `IEntityModelSet` 接口的非泛型版本，返回 `IReadOnlyList<IEntityModel>`。本功能旨在添加泛型版本支持，允许用户通过类型安全的泛型方法获取指定类型的模型。

### 重要约束
- 必须保持与现有 `MoSetAttribute` 的向后兼容性
- 必须保持现有生成的代码不变
- 泛型实现应避免运行时反射，确保 AOT 编译友好
- 泛型方法仅提供以下两个，不额外添加其他方法

## 技术背景

### 现有实现

**现有特性**：
```csharp
/// <summary>
/// 建模集合特性，标记类以启用源代码生成自动收集功能
/// </summary>
[AttributeUsage(AttributeTargets.Class)]
public class MoSetAttribute : Attribute
{
}
```

**现有接口**：
```csharp
/// <summary>
/// 实体模型集合接口，用于建模信息的自动收集
/// </summary>
public interface IEntityModelSet
{
    /// <summary>
    /// 获取集合中的所有实体模型
    /// </summary>
    /// <returns>所有实体模型</returns>
    System.Collections.Generic.IReadOnlyList<IEntityModel> GetModels();

    /// <summary>
    /// 获取集合中的模型数量
    /// </summary>
    int Count { get; }
}
```

**现有使用方式**：
```csharp
[MoSet]
public partial class UserSet
{
}
```

生成的代码：
```csharp
public partial class UserSet : IEntityModelSet
{
    private static readonly IEntityModel[] _models;

    static UserSet()
    {
        _models = new IEntityModel[]
        {
            User.GetEntityModel(),
            Order.GetEntityModel(),
        };
    }

    public System.Collections.Generic.IReadOnlyList<IEntityModel> GetModels()
    {
        return _models;
    }

    public int Count => _models.Length;
}
```

### 新增需求

为标记了 `MoSet` 特性的类添加类型安全的泛型方法，通过 `typeof` 和 `switch case` 实现编译时类型判断：

```csharp
[MoSet]
public partial class UserSet
{
}
```

生成的代码应包含：
```csharp
#if !NETSTANDARD2_0
#nullable enable
#endif

public partial class UserSet : IEntityModelSet
{
    private static readonly IEntityModel[] _models;
    private static readonly System.Type _userType = typeof(User);
    private static readonly System.Type _orderType = typeof(Order);

    static UserSet()
    {
        _models = new IEntityModel[]
        {
            User.GetEntityModel(),
            Order.GetEntityModel(),
        };
    }

    public System.Collections.Generic.IReadOnlyList<IEntityModel> GetModels()
    {
        return _models;
    }

    public int Count => _models.Length;

    /// <summary>
    /// 获取指定类型的模型
    /// </summary>
    /// <typeparam name="T">模型类型</typeparam>
    /// <returns>指定类型的模型</returns>
    /// <exception cref="System.NotSupportedException">当类型不匹配时抛出</exception>
    public IEntityModel GetModel<T>() where T : class
    {
        return typeof(T) switch
        {
            var t when t == _userType => User.GetEntityModel(),
            var t when t == _orderType => Order.GetEntityModel(),
            _ => throw new System.NotSupportedException($"不支持的模型类型: {typeof(T).FullName}")
        };
    }

    /// <summary>
    /// 尝试获取指定类型的模型
    /// </summary>
    /// <typeparam name="T">模型类型</typeparam>
    /// <returns>模型对象，未找到时返回 null</returns>
#if !NETSTANDARD2_0
    public IEntityModel? TryGetModel<T>() where T : class
#else
    public IEntityModel TryGetModel<T>() where T : class
#endif
    {
        return typeof(T) switch
        {
            var t when t == _userType => User.GetEntityModel(),
            var t when t == _orderType => Order.GetEntityModel(),
            _ => null
        };
    }
}
```

## 实现方案

### 1. 具体步骤

1. **保持 MoSetAttribute 不变**
   - 不修改现有特性定义
   - 确保向后兼容性

2. **修改 ModelSetSyntaxReceiver**
   - 在接收器中收集所有标记了 `MoTable` 或 `MoQuery` 的类
   - 提取每个类的命名空间和类名

3. **更新 ModelSetSourceGenerator**
   - 为每个收集到的模型类生成静态 `Type` 字段
   - 生成 `GetModel<T>()` 方法，使用 `switch case` 匹配类型
   - 生成 `TryGetModel<T>()` 方法，返回 null 或模型对象
   - 处理不同 .NET 版本的 nullable 兼容性

4. **添加单元测试**
   - 测试现有功能保持不变
   - 测试 `GetModel<T>()` 正确返回指定类型
   - 测试 `GetModel<T>()` 对未知类型抛出异常
   - 测试 `TryGetModel<T>()` 返回正确值或 null

5. **更新文档**
   - 更新用户指南，说明泛型用法

## 使用示例

### 示例 1：现有方式保持不变

```csharp
[MoSet]
public partial class AllModels
{
}
```

```csharp
var models = AllModels.GetModels();
foreach (var model in models)
{
    Console.WriteLine(model.FullName);
}
```

### 示例 2：泛型方式获取模型

```csharp
[MoSet]
public partial class TypedModels
{
}
```

```csharp
// 获取所有模型
var allModels = TypedModels.GetModels();

// 通过泛型获取模型
var userModel = TypedModels.GetModel<User>();
Console.WriteLine(userModel.FullName);

// 使用 TryGetModel 避免
var user = TypedModels.TryGetModel<User>();
if (user != null)
{
    Console.WriteLine(user.FullName);
}
```
