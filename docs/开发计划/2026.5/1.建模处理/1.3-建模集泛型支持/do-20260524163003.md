- 保持与现有 `MoSetAttribute` 的向后兼容性
- 保持现有生成的代码不变
- 泛型实现避免运行时反射，使用 `typeof` 和 `switch case` 实现编译时类型判断
- 仅提供两个泛型方法：`GetModel<T>()` 和 `TryGetModel<T>()`
- 处理不同 .NET 版本的 nullable 兼容性

## 文件修改详情

### 1. Delly.Modeling.Generator/ModelSetSourceGenerator.cs

#### 变更点 1：添加 using 指令生成
为有命名空间的模型类添加 using 指令，避免命名空间引用错误。

```csharp
// 为有命名空间的模型类添加 using 指令
var uniqueNamespaces = modelClasses
    .Where(m => !string.IsNullOrEmpty(m.Namespace))
    .Select(m => m.Namespace)
    .Distinct()
    .ToList();

foreach (var ns in uniqueNamespaces)
{
    sb.AppendLine($"using {ns};");
}
```

#### 变更点 2：生成静态 Type 字段
为每个模型类生成静态 Type 字段，用于编译时类型匹配。

```csharp
// 为每个模型类生成静态 Type 字段
foreach (var modelInfo in modelClasses)
{
    var fieldName = GetFieldName(modelInfo.ClassName);
    sb.AppendLine($"    private static readonly System.Type _{fieldName} = typeof({modelInfo.ClassName});");
}
```

#### 变更点 3：生成 GetModel<T>() 方法
使用 switch case 实现类型匹配，避免运行时反射。

```csharp
sb.AppendLine("    public IEntityModel GetModel<T>() where T : class");
sb.AppendLine("    {");
sb.AppendLine("        return typeof(T) switch");
sb.AppendLine("        {");
foreach (var modelInfo in modelClasses)
{
    var fieldName = GetFieldName(modelInfo.ClassName);
    sb.AppendLine($"            var t when t == _{fieldName} => {modelInfo.ClassName}.GetEntityModel(),");
}
sb.AppendLine("            _ => throw new System.NotSupportedException($\"不支持的模型类型: {typeof(T).FullName}\")");
sb.AppendLine("        };");
sb.AppendLine("    }");
```

#### 变更点 4：生成 TryGetModel<T>() 方法
处理 nullable 兼容性，为 .NET Standard 2.0 和 .NET 6.0+ 生成不同的返回类型。

```csharp
sb.AppendLine("#if !NETSTANDARD2_0");
sb.AppendLine("    public IEntityModel? TryGetModel<T>() where T : class");
sb.AppendLine("#else");
sb.AppendLine("    public IEntityModel TryGetModel<T>() where T : class");
sb.AppendLine("#endif");
sb.AppendLine("    {");
sb.AppendLine("        return typeof(T) switch");
sb.AppendLine("        {");
foreach (var modelInfo in modelClasses)
{
    var fieldName = GetFieldName(modelInfo.ClassName);
    sb.AppendLine($"            var t when t == _{fieldName} => {modelInfo.ClassName}.GetEntityModel(),");
}
sb.AppendLine("            _ => null");
sb.AppendLine("        };");
sb.AppendLine("    }");
```

#### 变更点 5：新增 GetFieldName 辅助方法
将类名转换为小写开头的字段名。

```csharp
private static string GetFieldName(string className)
{
    if (string.IsNullOrEmpty(className))
        return className;

    return char.ToLower(className[0]) + className.Substring(1);
}
```

## 生成的代码示例

最终生成的代码（`EntityModelSet.ModelSet.g.cs`）：

```csharp
#if !NETSTANDARD2_0
#nullable enable
#endif

using System;
using Delly.Modeling;
using Demo;

public partial class EntityModelSet : IEntityModelSet
{
    private static readonly IEntityModel[] _models;
    private static readonly System.Type _userEntity = typeof(UserEntity);
    private static readonly System.Type _userQuery = typeof(UserQuery);

    static EntityModelSet()
    {
        _models = new IEntityModel[]        {
            UserEntity.GetEntityModel(),
            UserQuery.GetEntityModel(),
        };
    }

    public System.Collections.Generic.IReadOnlyList<IEntityModel> GetModels()
    {
        return _models;
    }

    public int Count => _models.Length;

    public IEntityModel GetModel<T>() where T : class
    {
        return typeof(T) switch
        {
            var t when t == _userEntity => UserEntity.GetEntityModel(),
            var t when t == _userQuery => UserQuery.GetEntityModel(),
            _ => throw new System.NotSupportedException($"不支持的模型类型: {typeof(T).FullName}")
        };
    }

#if !NETSTANDARD2_0
    public IEntityModel? TryGetModel<T>() where T : class
#else
    public IEntityModel TryGetModel<T>() where T : class
#endif
    {
        return typeof(T) switch
        {
            var t when t == _userEntity => UserEntity.GetEntityModel(),
            var t when t == _userQuery => UserQuery.GetEntityModel(),
            _ => null
        };
    }
}
```
