# 1.3 建模集泛型支持

## 任务信息
- **任务号**: 1.3
- **任务名称**: 建模集泛型支持
- **责任人**: 郑琳
- **任务类型**: 功能开发
- **任务描述**: 在建模集源生成中添加泛型转换支持

## 功能概述

建模集（`MoSetAttribute`）目前只支持生成实现 `IEntityModelSet` 接口的非泛型版本，返回 `IReadOnlyList<IEntityModel>`。本功能旨在添加泛型版本支持，允许用户通过指定基类型来获取类型安全的模型集合。

### 重要约束
- 必须保持与现有 `MoSetAttribute` 的向后兼容性
- 泛型支持应通过可选的方式提供，不强制使用泛型
- TODO: 确认泛型特性的命名方式（建议：`MoSetAttribute` 增加可选的基类型参数）

## 技术背景

### 现有实现

**现有特性**：
```csharp
/// <summary>
/// 建模集合特性，标记类以启用源代码生成自动收集功能
/// </summary>
[AttributeUsage(AttributeTargets.Class)]
public class MoSetAttribute : Attribute
{
}
```

**现有接口**：
```csharp
/// <summary>
/// 实体模型集合接口，用于建模信息的自动收集
/// </summary>
public interface IEntityModelSet
{
    /// <summary>
    /// 获取集合中的所有实体模型
    /// </summary>
    /// <returns>所有实体模型</returns>
    System.Collections.Generic.IReadOnlyList<IEntityModel> GetModels();

    /// <summary>
    /// 获取集合中的模型数量
    /// </summary>
    int Count { get; }
}
```

**现有使用方式**：
```csharp
[MoSet]
public partial class UserSet
{
}
```

生成的代码：
```csharp
public partial class UserSet : IEntityModelSet
{
    private static readonly IEntityModel[] _models;

    static UserSet()
    {
        _models = new IEntityModel[]
        {
            User.GetEntityModel(),
            Order.GetEntityModel(),
        };
    }

    public System.Collections.Generic.IReadOnlyList<IEntityModel> GetModels()
    {
        return _models;
    }

    public int Count => _models.Length;
}
```

### 新增需求

允许用户通过特性参数指定基类型，生成类型安全的泛型版本：

```csharp
[MoSet(typeof(IEntityModel))]
public partial class UserSet
{
}
```

生成的代码应包含泛型方法：
```csharp
public partial class UserSet : IEntityModelSet
{
    // ... 现有代码保持不变 ...

    /// <summary>
    /// 获取指定类型的所有模型
    /// </summary>
    /// <typeparam name="T">模型类型</typeparam>
    /// <returns>指定类型的所有模型</returns>
    public IReadOnlyList<T> GetModels<T>() where T : IEntityModel
    {
        return _models.OfType<T>().ToArray();
    }
}
```

### 现有接口

相关接口定义：
```csharp
/// <summary>
/// 实体模型接口
/// </summary>
public interface IEntityModel
{
    /// <summary>
    /// 获取类全称
    /// </summary>
    string FullName { get; }

    /// <summary>
    /// 获取表名
    /// </summary>
    string TableName { get; }

    /// <summary>
    /// 获取所有属性信息
    /// </summary>
    /// <returns>属性信息列表</returns>
    IReadOnlyList<IEntityProperty> GetProperties();
}
```

## 使用示例

### 示例 1：非泛型版本（现有方式）

```csharp
[MoSet]
public partial class AllModels
{
}
```

```csharp
var models = AllModels.GetModels();
foreach (var model in models)
{
    Console.WriteLine(model.FullName);
}
```

### 示例 2：泛型版本（新增方式）

```csharp
[MoSet(typeof(IEntityModel))]
public partial class TypedModels
{
}
```

```csharp
// 获取所有模型
var allModels = TypedModels.GetModels();

// 获取特定类型的模型
var userModels = TypedModels.GetModels<IUserModel>();
```
