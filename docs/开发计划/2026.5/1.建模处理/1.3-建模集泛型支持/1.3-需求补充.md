# 1.3 需求补充

本文档用于记录任务 1.3 建模集泛型支持的需求补充信息。

## 补充事项

### 待补充

1. 保持现有代码不变

2. 泛型实现案例（需要进行评估） 
   - 借助typeof实现类型判断
    ```

    private static Type _userType = typeof(User) 

    /// <summary>
    /// 获取指定类型的所有模型
    /// </summary>
    /// <typeparam name="T">模型类型</typeparam>
    /// <returns>指定类型的所有模型</returns>
    public IEntityModel GetModel<T>()
    {
        switch(typeof(T))
        {
            case _userType
                return User.GetEntityModel();
            default:
                // 抛出异常
        }
    }
    ```
3. 泛型方式获取建模，只有以下两个，不需要额外添加其他方法
   - `IEntityModel GetModel<T>() where T : class`
   - `IEntityModel? TryGetModel<T>() where T : class`
     - 需要考虑不同.net版本的nullable兼容性

4. 需要重新执行生成
   - `IEntityModelSet` 中需要添加 `GetModel` 和 `TryGetModel` 的定义
