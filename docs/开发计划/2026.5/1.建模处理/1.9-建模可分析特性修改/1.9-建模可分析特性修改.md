# 1.9 建模可分析特性修改

## 任务信息

- **任务号**: 1.9
- **任务名称**: 建模可分析特性修改
- **责任人**: 郑琳
- **任务类型**: 需求变更
- **任务描述**: 修改 `ParsableAttribute` 实现在目标类上通过定义 `[Parsable(typeof(UserParser))]` 方式实现对 `IParsable<T>` 实现类的源生成支持

## 功能概述

修改 `ParsableAttribute` 特性，从原来的无参数标记特性改为接受 `Type` 参数的参数化特性。使用方式从在 Parser 类上标记 `[Parsable]` 改为在目标类上标记 `[Parsable(typeof(ParserClass))]`。源生成器逻辑相应调整为从目标类读取特性，直接获取 Parser 类型，而非扫描所有标记了特性的 Parser 类。

**重要约束**：
- 所有 `interface` 定义不允许使用 `static abstract`
- 本项目不允许出现反射方式调用！！！
- 接口定义使用非空类型 `object`，实现类内部可返回 `null`
- `ParsableAttribute` 现在接受 `Type` 参数，标记在目标类上
- 源生成器从目标类读取特性，获取 Parser 类型
- 不同 .NET 版本使用条件编译提供支持
- `IParsable<T>` 实现类必须提供无参构造函数

## 技术背景

### 现有实现

**原 ParsableAttribute（1.8任务）**：

```csharp
/// <summary>
/// 用于标记 IParsable<T> 实现类的特性，供源生成器查找使用
/// </summary>
[AttributeUsage(AttributeTargets.Class, AllowMultiple = false, Inherited = false)]
public sealed class ParsableAttribute : Attribute
{
}
```

**原使用方式**：

```csharp
// Parser 类上标记 [Parsable] 特性
[Parsable]
public class UserParser : IParsable<User>
{
    public object? TryParse(object? obj)
    {
        // 解析逻辑...
    }
}

// 目标类无需标记
[Modelable]
public partial class User
{
    public long Id { get; set; }
    public string Name { get; set; }
}
```

**原源生成器查找逻辑**：
- 扫描程序集中所有标记了 `[Parsable]` 的类
- 验证该类是否实现了 `IParsable<T>` 接口
- 从 `IParsable<T>` 的泛型类型参数中提取目标类型 T
- 建立目标类型与 Parser 类的映射关系

### 变更需求

**新 ParsableAttribute**：

```csharp
/// <summary>
/// 用于标记目标类并指定其 Parser 类型的特性，供源生成器查找使用
/// </summary>
/// <param name="parserType">Parser 类型，必须实现 IParsable&lt;T&gt; 接口，其中 T 为目标类型</param>
[AttributeUsage(AttributeTargets.Class, AllowMultiple = false, Inherited = false)]
public sealed class ParsableAttribute : Attribute
{
    /// <summary>
    /// Parser 类型
    /// </summary>
    public Type ParserType { get; }

    /// <summary>
    /// 初始化 ParsableAttribute 实例
    /// </summary>
    /// <param name="parserType">Parser 类型</param>
    public ParsableAttribute(Type parserType)
    {
        ParserType = parserType;
    }
}
```

**新使用方式**：

```csharp
// Parser 类无需标记特性
public class UserParser : IParsable<User>
{
    public object? TryParse(object? obj)
    {
        // 解析逻辑...
    }
}

// 目标类上标记 [Parsable(typeof(ParserClass))]
[Parsable(typeof(UserParser))]
[Modelable]
public partial class User
{
    public long Id { get; set; }
    public string Name { get; set; }
}
```

**新源生成器查找逻辑**：
- 遍历所有标记了特性（如 `[Modelable]`、`[MoTable]`、`[MoQuery]`）的目标类
- 检查目标类是否标记了 `[Parsable(typeof(ParserClass))]`
- 如果标记了，从特性的 `ParserType` 属性获取 Parser 类型
- 验证 Parser 类型是否实现了 `IParsable<T>` 接口
- 验证 `IParsable<T>` 的泛型参数 T 是否为目标类类型

## 实现方案

### 1. 修改 ParsableAttribute 特性

```csharp
#if !NETSTANDARD2_0
#nullable enable
#endif

using System;

namespace Delly.Modeling
{
    /// <summary>
    /// 用于标记目标类并指定其 Parser 类型的特性，供源生成器查找使用
    /// </summary>
    /// <remarks>
    /// 标记此特性的目标类可通过源生成器生成的 Parse/TryParse 方法进行解析。
    /// 指定的 Parser 类型必须实现 IParsable&lt;T&gt; 接口，其中 T 为目标类类型。
    /// </remarks>
    [AttributeUsage(AttributeTargets.Class, AllowMultiple = false, Inherited = false)]
    public sealed class ParsableAttribute : Attribute
    {
        /// <summary>
        /// Parser 类型
        /// </summary>
        public Type ParserType { get; }

        /// <summary>
        /// 初始化 ParsableAttribute 实例
        /// </summary>
        /// <param name="parserType">Parser 类型，必须实现 IParsable&lt;T&gt; 接口</param>
        public ParsableAttribute(Type parserType)
        {
            ParserType = parserType;
        }
    }
}
```

### 2. 修改源生成器查找逻辑

在以下源生成器中添加查找逻辑：

| 源生成器 | 特性 | 需要修改 |
|:--------|:----|:--------|
| `ModelableSourceGenerator` | `ModelableAttribute` | 添加 Parser 查找逻辑 |
| `ModelTableSourceGenerator` | `MoTableAttribute` | 添加 Parser 查找逻辑 |
| `ModelTableSourceGenerator` | `MoQueryAttribute` | 添加 Parser 查找逻辑 |

**源生成器查找逻辑**：

```csharp
// 在处理目标类时查找 Parser 类型
private static Type? FindParserType(INamedTypeSymbol targetClass, GeneratorExecutionContext context)
{
    // 查找目标类上的 [Parsable(typeof(ParserClass))] 特性
    var parsableAttr = targetClass.GetAttributes()
        .FirstOrDefault(a => a.AttributeClass?.Name == "ParsableAttribute");

    if (parsableAttr == null || parsableAttr.ConstructorArguments.Length == 0)
        return null;

    // 获取 Parser 类型
    var parserTypeSymbol = parsableAttr.ConstructorArguments[0].Value as INamedTypeSymbol;
    if (parserTypeSymbol == null)
        return null;

    // 验证 Parser 类型是否实现了 IParsable<T>
    bool foundMatchingInterface = false;
    foreach (var iface in parserTypeSymbol.AllInterfaces)
    {
        if (iface.Name == "IParsable" && iface is INamedTypeSymbol namedInterface &&
            namedInterface.IsGenericType && namedInterface.TypeArguments.Length == 1)
        {
            var targetTypeArgument = namedInterface.TypeArguments[0];
            // 验证泛型参数 T 是否为目标类类型
            if (SymbolEqualityComparer.Default.Equals(targetTypeArgument, targetClass))
            {
                foundMatchingInterface = true;
                break;
            }
        }
    }

    if (!foundMatchingInterface)
    {
        // 报告诊断错误：Parser 类型未实现 IParsable<目标类>
        context.ReportDiagnostic(Diagnostic.Create(
            new DiagnosticDescriptor(
                "DM002",
                "Invalid Parser Type",
                "Parser type '{0}' must implement IParsable<{1}> interface",
                "Usage",
                DiagnosticSeverity.Error,
                true),
            targetClass.Locations.FirstOrDefault(),
            parserTypeSymbol.Name,
            targetClass.Name));
        return null;
    }

    return parserTypeSymbol;
}
```

### 3. 生成的 TryParse 函数逻辑

**生成的 `TryParse` 函数逻辑**（非 .NET Standard 2.0）：

```csharp
// Parser 实例缓存
private static IParsable<{ClassName}>? _parser;

/// <summary>
/// 尝试将输入对象解析为目标类型实例
/// </summary>
/// <param name="obj">输入对象</param>
/// <returns>目标类型实例，解析失败时返回 null</returns>
public override object? TryParse(object? obj)
{
    if (obj == null)
        return null;

    if (obj is {ClassName} target)
        return target;

    // 获取或创建 Parser 实例
    if (_parser == null)
        _parser = new {ParserClassName}();

    return _parser.TryParse(obj);
}
```

**生成的 `TryParse` 函数逻辑**（.NET Standard 2.0）：

```csharp
// Parser 实例缓存
private static IParsable<{ClassName}> _parser;

/// <summary>
/// 尝试将输入对象解析为目标类型实例
/// </summary>
/// <param name="obj">输入对象</param>
/// <returns>目标类型实例，解析失败时返回 null</returns>
public override object TryParse(object obj)
{
    if (obj == null)
        return null;

    if (obj is {ClassName} target)
        return target;

    // 获取或创建 Parser 实例
    if (_parser == null)
        _parser = new {ParserClassName}();

    return _parser.TryParse(obj);
}
```

### 4. 生成的 Parse 函数逻辑

**生成的 `Parse` 函数逻辑**（非 .NET Standard 2.0）：

```csharp
/// <summary>
/// 将输入对象解析为目标类型实例
/// </summary>
/// <param name="obj">输入对象</param>
/// <returns>目标类型实例</returns>
/// <exception cref="ArgumentException">当对象无法转换为目标类型时抛出</exception>
public override object Parse(object? obj)
{
    var result = TryParse(obj);
    if (result == null)
        throw new ArgumentException($"Cannot convert {obj?.GetType().Name ?? "null"} to {ClassName}");
    return result;
}
```

**生成的 `Parse` 函数逻辑**（.NET Standard 2.0）：

```csharp
/// <summary>
/// 将输入对象解析为目标类型实例
/// </summary>
/// <param name="obj">输入对象</param>
/// <returns>目标类型实例</returns>
/// <exception cref="ArgumentException">当对象无法转换为目标类型时抛出</exception>
public override object Parse(object obj)
{
    var result = TryParse(obj);
    if (result == null)
        throw new ArgumentException($"Cannot convert {(obj != null ? obj.GetType().Name : "null")} to {ClassName}");
    return result;
}
```

### 5. 用户使用示例

```csharp
// 定义 Parser 类（无需标记特性）
public class UserParser : IParsable<User>
{
    public object? TryParse(object? obj)
    {
        if (obj == null)
            return null;

        if (obj is User user)
            return user;

        if (obj is long id)
            return new User { Id = id, Name = string.Empty };

        if (obj is string str && long.TryParse(str, out var longId))
            return new User { Id = longId, Name = string.Empty };

        return null;
    }
}

// 在目标类上标记 [Parsable(typeof(ParserClass))]
[Parsable(typeof(UserParser))]
[Modelable]
public partial class User
{
    public long Id { get; set; }
    public string Name { get; set; }
}

// 使用生成的解析方法
var userModel = User.GetEntityModel();
var user = (User)userModel.Parse(123L);
Console.WriteLine(user.Id);  // 输出: 123

var user2 = (User)userModel.TryParse("456");
Console.WriteLine(user2?.Id);  // 输出: 456

// 多个目标类型共享同一个 Parser
public class LongParser : IParsable<long>
{
    public object? TryParse(object? obj)
    {
        if (obj == null)
            return null;

        if (obj is long value)
            return value;

        if (obj is int intValue)
            return intValue;

        if (obj is string str && long.TryParse(str, out var longValue))
            return longValue;

        return null;
    }
}

[Parsable(typeof(LongParser))]
[Modelable]
public partial class OrderId
{
    public long Value { get; set; }
}

[Parsable(typeof(LongParser))]
[Modelable]
public partial class UserId
{
    public long Value { get; set; }
}
```
