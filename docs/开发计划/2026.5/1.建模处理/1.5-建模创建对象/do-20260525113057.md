## 备注

本次返工完成了以下变更：

1. **接口签名变更**: `CreateInstance(object[] args)` → `CreateInstance(params object[] args)`
2. **参数变为可选参数**: 使用 `params` 关键字，允许直接传递参数而无需创建数组
3. **调用方式改进**: 可以使用 `CreateInstance(arg1, arg2)` 而非 `CreateInstance(new[] { arg1, arg2 })`
4. **兼容性保持**: 内部实现仍处理空值和数组参数，保持向后兼容

## 使用方式对比

### 改进前
```csharp
// 无参创建
var instance1 = model.CreateInstance(new object[0]);

// 带参数创建
var instance2 = model.CreateInstance(new object[] { "arg1", "arg2" });

// 内置模型类
var value = stringModel.CreateInstance(new object[] { "Hello" });
```

### 改进后
```csharp
// 无参创建（直接调用）
var instance1 = model.CreateInstance();

// 带参数创建（直接传递参数）
var instance2 = model.CreateInstance("arg1", "arg2");

// 内置模型类（直接传递参数）
var value = stringModel.CreateInstance("Hello");
```
