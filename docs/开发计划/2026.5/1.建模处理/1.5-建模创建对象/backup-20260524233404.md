# 1.5 建模创建对象

## 任务信息
- **任务号**: 1.5
- **任务名称**: 建模创建对象
- **责任人**: 郑琳
- **任务类型**: 功能开发
- **任务描述**: 添加建模直接创建对象实例的能力

## 功能概述

为 `[Modelable]` 和 `[MoTable]`/`[MoQuery]` 类添加源生成能力，使其能够通过模型对象直接创建新实例。此功能通过在 `IBaseModel` 接口中添加 `CreateInstance()` 方法，允许用户通过模型对象创建新实例。

**重要约束**：
- 必须保持与现有功能的向后兼容性
- 生成的代码需要支持 AOT 编译
- `CreateInstance()` 返回 `object` 类型，需要使用者进行类型转换
- 同时兼容 `IModel` 与 `IEntityModel` 的源生成处理
- **使用建模特性的类不允许定义带参数的构造函数**，仅支持无参构造函数（显式或隐式）

## 技术背景

### 现有实现

**Modelable 特性**：
```csharp
[AttributeUsage(AttributeTargets.Class, Inherited = false, AllowMultiple = false)]
public sealed class ModelableAttribute : Attribute
{
}
```

**MoTable/MoQuery 特性**：
```csharp
[AttributeUsage(AttributeTargets.Class, Inherited = false, AllowMultiple = false)]
public sealed class MoTableAttribute : Attribute
{
    public string? Name { get; }
}

[AttributeUsage(AttributeTargets.Class, Inherited = false, AllowMultiple = false)]
public sealed class MoQueryAttribute : Attribute
{
}
```

**现有使用方式**：
```csharp
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
}

// 通过静态方法获取模型
var model = User.GetModel();
var properties = model.GetProperties();
```

```csharp
[MoTable("users")]
public partial class User
{
    [MoColumn(Key = true, AutoIncrement = true)]
    public long Id { get; set; }

    [MoColumn]
    public string Name { get; set; } = string.Empty;
}

// 通过静态方法获取实体模型
var entityModel = User.GetEntityModel();
var properties = entityModel.GetProperties();
```

**现有生成的模型结构**：
```csharp
// ModelableSourceGenerator 生成的模型类
public class mc__User : IModel
{
    public static mc__User Instance => _instance;
    public string Name => "UserModel";
    public string Namespace => "...";
    public IModelProperty[] GetProperties() { ... }
    public IModelProperty? GetProperty(string name) { ... }
}

// ModelTableSourceGenerator 生成的实体模型类
public class emc__User : IEntityModel
{
    public static emc__User Instance => _instance;
    public string Name => "users";
    public string ClassName => "User";
    public string Namespace => "...";
    public IEntityModelProperty[] GetProperties() { ... }
    public IEntityModelProperty? GetProperty(string name) { ... }
    public EntityModelIndex[] GetIndexes() { ... }
}
```

### 3. 使用约束

**不允许的用法**（带参数的构造函数）：
```csharp
// ❌ 错误 - 主构造函数带参数
[Modelable]
public partial class User(string id)  // 不允许
{
    public string Id { get; } = id;
}

// ❌ 错误 - 构造函数带参数
[Modelable]
public partial class User
{
    public User(string id)  // 不允许
    {
        Id = id;
    }
    public string Id { get; set; } = string.Empty;
}

// ❌ 错误 - MoTable 带参数构造函数
[MoTable("users")]
public partial class User(string id)  // 不允许
{
    public string Id { get; } = id;
}
```

## 使用示例

### 示例 1：Modelable 类

```csharp
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
    public string Name { get; set; } = string.Empty;
}

// 通过模型对象创建实例
var model = User.GetModel();
var user = (User)model.CreateInstance();

user.Id = "1";
user.Name = "张三";
```

### 示例 2：MoTable 类

```csharp
[MoTable("users")]
public partial class User
{
    [MoColumn(Key = true, AutoIncrement = true)]
    public long Id { get; set; }

    [MoColumn]
    public string Name { get; set; } = string.Empty;
}

// 通过实体模型对象创建实例
var entityModel = User.GetEntityModel();
var user = (User)entityModel.CreateInstance();

user.Name = "张三";
```

### 示例 3：MoQuery 类

```csharp
[MoQuery]
public partial class UserQuery
{
    public string? Name { get; set; }
    public int? MinAge { get; set; }
}

// 通过实体模型对象创建实例
var entityModel = UserQuery.GetEntityModel();
var query = (UserQuery)entityModel.CreateInstance();

query.Name = "张三";
query.MinAge = 18;
```

### 示例 4：批量创建

```csharp
var userModel = User.GetEntityModel();
var users = new List<User>();
for (int i = 0; i < 10; i++)
{
    var user = (User)userModel.CreateInstance();
    user.Name = $"用户{i}";
    users.Add(user);
}
```

### 示例 5：Demo 编写结构

```csharp
// CreateInstanceDemo.cs
public class CreateInstanceDemo
{
    public void Run()
    {
        // Modelable 示例
        var userModel = User.GetModel();
        var user = (User)userModel.CreateInstance();
        user.Id = "1";
        user.Name = "张三";

        // MoTable 示例
        var entityModel = User.GetEntityModel();
        var entityUser = (User)entityModel.CreateInstance();
        entityUser.Name = "李四";

        // MoQuery 示例
        var queryModel = UserQuery.GetEntityModel();
        var query = (UserQuery)queryModel.CreateInstance();
        query.Name = "王五";
        query.MinAge = 18;

        Console.WriteLine($"User: {user.Id} - {user.Name}");
        Console.WriteLine($"EntityUser: {entityUser.Name}");
        Console.WriteLine($"Query: {query.Name} - {query.MinAge}");
    }
}

// Program.cs
var demo = new CreateInstanceDemo();
demo.Run();
```
