# 1.5 建模创建对象

- **责任人**: 郑琳
- **任务类型**: 功能开发
- **任务描述**: 添加建模直接创建对象实例的能力

## 功能概述

为 `[Modelable]` 类添加源生成能力，使其能够通过模型对象直接创建新实例。此功能通过在 `IBaseModel` 接口中添加 `CreateInstance()` 方法，允许用户通过模型对象创建新实例。

**重要约束**：
- 必须保持与现有功能的向后兼容性
- 生成的代码需要支持 AOT 编译
- `CreateInstance()` 返回 `object` 类型，需要使用者进行类型转换
- 使用 `new T()` 语法创建实例，不支持参数化构造函数

## 技术背景

### 现有实现

**Modelable 特性**：
```csharp
[AttributeUsage(AttributeTargets.Class, Inherited = false, AllowMultiple = false)]
public sealed class ModelableAttribute : Attribute
{
}
```

**现有使用方式**：
```csharp
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
    public string Name { get; set; } = string.Empty;
    public string? Password { get; set; }
}

// 通过静态方法获取模型
var model = User.GetModel();

// 通过模型访问属性
var properties = model.GetProperties();
var idProperty = model.GetProperty(nameof(User.Id));
```

**现有生成的模型结构**：
```csharp
// 生成的模型类
public class mc__User : IModel
{
    public static mc__User Instance => _instance;
    public string Name => "UserModel";
    public string Namespace => "...";
    public IModelProperty[] GetProperties() { ... }
    public IModelProperty? GetProperty(string name) { ... }
}

// User 类中的静态方法
public partial class User
{
    public static mc__User GetModel() => mc__User.Instance;
}
```

### 新增需求

```csharp

```

## 使用示例

### 示例 1：基本使用

```csharp
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
    public string Name { get; set; } = string.Empty;
    public string? Password { get; set; }
}

// 通过模型对象创建实例
var model = User.GetModel();
var user = (User)model.CreateInstance();

user.Id = "1";
user.Name = "张三";
```

### 示例 2：主构造函数类

```csharp
[Modelable]
public partial class User(string id)
{
    public string Id { get; } = id;
    public string Name { get; set; } = string.Empty;
}

// 通过模型对象创建实例（使用默认参数值）
var model = User.GetModel();
var user = (User)model.CreateInstance();

// Id 为 string.Empty（或 TODO 中确定的默认值）
user.Name = "张三";
```

### 示例 3：多模型创建

```csharp
var userModel = User.GetModel();
var orderModel = Order.GetModel();

// 批量创建实例
var users = new List<User>();
for (int i = 0; i < 10; i++)
{
    var user = (User)userModel.CreateInstance();
    user.Id = $"user-{i}";
    users.Add(user);
}
```
