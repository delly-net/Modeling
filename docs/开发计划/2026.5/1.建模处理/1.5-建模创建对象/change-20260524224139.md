# 变更记录 - 1.5 建模创建对象

## 备份信息

**原始文档备份**: [backup-20260524224139.md](./backup-20260524224139.md)

## 变更内容

### 1. 新增接口方法

**IBaseModel 接口新增方法**：
```csharp
/// <summary>
/// 创建模型类型的新实例
/// </summary>
/// <returns>模型类型的新实例</returns>
object CreateInstance();
```

### 2. 源生成扩展

**ModelableSourceGenerator 新增生成内容**：
- 在生成的 `mc__{ClassName}` 模型类中实现 `CreateInstance()` 方法
- 方法返回 `new {ClassName}()` 创建的新实例

**生成示例**：
```csharp
// 原始代码
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
}

// 生成的模型类实现
public class mc__User : IModel
{
    // ... 现有成员 ...

    /// <summary>
    /// 创建模型类型的新实例
    /// </summary>
    /// <returns>模型类型的新实例</returns>
    public object CreateInstance()
    {
        return new User();
    }
}
```

## 变更原因

根据需求补充文档的反馈，需要通过 `IBaseModel` 接口提供统一的创建对象能力，使外部代码可以通过模型对象创建新实例，而无需知道具体类型。

## 可能产生的影响

1. **接口变更**：`IBaseModel` 接口新增方法，所有实现类需要实现该方法
2. **现有模型类**：需要更新 `ModelableSourceGenerator` 以生成 `CreateInstance()` 方法
3. **AOT 兼容性**：`CreateInstance()` 使用 `new T()` 语法，AOT 友好
4. **类型安全**：返回 `object` 类型，需要使用者进行类型转换

## 差异点总结

| 项目 | 原始设计 | 新设计 |
|:-----|:---------|:-------|
| 创建方式 | 静态方法 `User.Create()` | 模型方法 `model.CreateInstance()` |
| 调用入口 | `User.Create()` | `User.GetModel().CreateInstance()` |
| 返回类型 | `User`（强类型） | `object`（弱类型） |
| 接口位置 | 静态扩展 | IBaseModel 接口方法 |
