- **任务描述**: 添加建模直接创建对象实例的能力

## 功能概述

为 `[Modelable]` 和 `[MoTable]`/`[MoQuery]` 类添加源生成能力，使其能够通过模型对象直接创建新实例。此功能通过在 `IBaseModel` 接口中添加 `CreateInstance(object?[] args)` 方法，允许用户通过模型对象创建新实例。

**重要约束**：
- 必须保持与现有功能的向后兼容性
- 生成的代码需要支持 AOT 编译
- `CreateInstance(object?[] args)` 返回 `object` 类型，需要使用者进行类型转换
- 同时兼容 `IModel` 与 `IEntityModel` 的源生成处理
- 支持无参构造函数和带参数构造函数
- 多构造函数情况下，出现参数数量相同的，需要抛出 `InvalidOperationException`

### 新增需求

**IBaseModel 接口新增方法**：
```csharp
/// <summary>
/// 模型对象基础接口
/// </summary>
public interface IBaseModel
{
    string Name { get; }
    string Namespace { get; }

    /// <summary>
    /// 创建模型类型的新实例
    /// </summary>
    /// <param name="args">构造函数参数数组</param>
    /// <returns>模型类型的新实例</returns>
    object CreateInstance(object?[] args);
}
```

**源生成实现（无参构造函数）**：
```csharp
public class mc__User : IModel
{
    public object CreateInstance(object?[] args)
    {
        return new User();
    }
}
```

**源生成实现（多构造函数）**：
```csharp
public class mc__User : IModel
{
    public object CreateInstance(object?[] args)
    {
        return args.Length switch
        {
            0 => new User(),
            1 => new User((string)args[0]),
            2 => new User((string)args[0], (int)args[1]),
            _ => throw new ArgumentException($"不支持 {args.Length} 个参数的构造函数")
        };
    }
}
```

### 2. 关键实现细节

**无参构造函数支持**：
```csharp
// 源代码
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
}

// 生成的代码
public object CreateInstance(object?[] args)
{
    return args.Length switch
    {
        0 => new User(),
        _ => throw new ArgumentException($"不支持 {args.Length} 个参数的构造函数")
    };
}
```

**单参数构造函数支持**：
```csharp
// 源代码
[Modelable]
public partial class User(string id)
{
    public string Id { get; } = id;
}

// 生成的代码
public object CreateInstance(object?[] args)
{
    return args.Length switch
    {
        1 => new User((string)args[0]),
        _ => throw new ArgumentException($"不支持 {args.Length} 个参数的构造函数")
    };
}
```

**多构造函数支持**：
```csharp
// 源代码
[Modelable]
public partial class User
{
    public User() { }
    public User(string id) { Id = id; }
    public User(string id, int age) { Id = id; Age = age; }
    public string Id { get; set; } = string.Empty;
    public int Age { get; set; }
}

// 生成的代码
public object CreateInstance(object?[] args)
{
    return args.Length switch
    {
        0 => new User(),
        1 => new User((string)args[0]),
        2 => new User((string)args[0], (int)args[1]),
        _ => throw new ArgumentException($"不支持 {args.Length} 个参数的构造函数")
    };
}
```

**参数冲突检测**：
```csharp
// 源代码 - 存在两个单参数构造函数
[Modelable]
public partial class User
{
    public User(string id) { Id = id; }  // 构造函数1
    public User(string name) { Name = name; }  // 构造函数2 - 参数数量相同
    public string Id { get; set; } = string.Empty;
    public string Name { get; set; } = string.Empty;
}

// 生成的代码 - 检测到冲突，抛出异常
public object CreateInstance(object?[] args)
{
    return args.Length switch
    {
        0 => throw new InvalidOperationException("存在 0 个参数的构造函数冲突"),
        1 => throw new InvalidOperationException("存在 1 个参数的构造函数冲突，无法确定使用哪一个"),
        _ => throw new ArgumentException($"不支持 {args.Length} 个参数的构造函数")
    };
}
```

**内置模型类实现**：
```csharp
// StringModel 实现
public class StringModel : IModel
{
    public object CreateInstance(object?[] args)
    {
        return args.Length switch
        {
            0 => string.Empty,
            1 => args[0]?.ToString() ?? string.Empty,
            _ => throw new ArgumentException($"不支持 {args.Length} 个参数的构造函数")
        };
    }
}

// Int32Model 实现
public class Int32Model : IModel
{
    public object CreateInstance(object?[] args)
    {
        return args.Length switch
        {
            0 => 0,
            1 => Convert.ToInt32(args[0]),
            _ => throw new ArgumentException($"不支持 {args.Length} 个参数的构造函数")
        };
    }
}
```

## 使用示例

### 示例 1：无参构造函数

```csharp
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
    public string Name { get; set; } = string.Empty;
}

// 通过模型对象创建实例
var model = User.GetModel();
var user = (User)model.CreateInstance(Array.Empty<object?>());

user.Id = "1";
user.Name = "张三";
```

### 示例 2：单参数构造函数

```csharp
[Modelable]
public partial class User(string id)
{
    public string Id { get; } = id;
    public string Name { get; set; } = string.Empty;
}

// 通过模型对象创建实例
var model = User.GetModel();
var user = (User)model.CreateInstance(new object?[] { "user-123" });

user.Name = "张三";
```

### 示例 3：多构造函数

```csharp
[Modelable]
public partial class Order
{
    public Order() { }
    public Order(string id) { Id = id; }
    public Order(string id, decimal amount) { Id = id; Amount = amount; }
    public string Id { get; set; } = string.Empty;
    public decimal Amount { get; set; }
}

// 无参创建
var model1 = Order.GetModel();
var order1 = (Order)model1.CreateInstance(Array.Empty<object?>());

// 单参创建
var model2 = Order.GetModel();
var order2 = (Order)model2.CreateInstance(new object?[] { "order-123" });

// 双参创建
var model3 = Order.GetModel();
var order3 = (Order)model3.CreateInstance(new object?[] { "order-456", 99.99m });
```

### 示例 4：MoTable 类

```csharp
[MoTable("users")]
public partial class User
{
    [MoColumn(Key = true, AutoIncrement = true)]
    public long Id { get; set; }

    [MoColumn]
    public string Name { get; set; } = string.Empty;
}

// 通过实体模型对象创建实例
var entityModel = User.GetEntityModel();
var user = (User)entityModel.CreateInstance(Array.Empty<object?>());

user.Name = "张三";
```

### 示例 5：批量创建

```csharp
var userModel = User.GetModel();
var users = new List<User>();
for (int i = 0; i < 10; i++)
{
    var user = (User)userModel.CreateInstance(new object?[] { $"user-{i}" });
    users.Add(user);
}
```

### 示例 6：Demo 编写结构

```csharp
// CreateInstanceDemo.cs
public class CreateInstanceDemo
{
    public void Run()
    {
        // 无参创建示例
        var userModel = User.GetModel();
        var user = (User)userModel.CreateInstance(Array.Empty<object?>());
        user.Id = "1";
        user.Name = "张三";

        // 单参创建示例
        var orderModel = Order.GetModel();
        var order = (Order)orderModel.CreateInstance(new object?[] { "order-123" });
        order.Amount = 99.99m;

        // 多参创建示例
        var order2 = (Order)orderModel.CreateInstance(new object?[] { "order-456", 199.99m });

        Console.WriteLine($"User: {user.Id} - {user.Name}");
        Console.WriteLine($"Order: {order.Id} - {order.Amount}");
        Console.WriteLine($"Order2: {order2.Id} - {order2.Amount}");
    }
}

// Program.cs
var demo = new CreateInstanceDemo();
demo.Run();
```

- `CreateInstance(object?[] args)` 能正确创建对象实例
- 支持无参构造函数类的对象创建
- 支持带参数构造函数类的对象创建
- 支持多构造函数的参数匹配
- 参数冲突时抛出 `InvalidOperationException`
- 不支持参数数量时抛出 `ArgumentException`
- 同时兼容 `IModel` 和 `IEntityModel` 的源生成
- ModelableSourceGenerator 生成的类支持 CreateInstance()
- ModelTableSourceGenerator 生成的类支持 CreateInstance()
- 返回的 `object` 可正确转换为对应类型
- 单元测试覆盖率 ≥ 80%
- Demo 按规范编写（功能特有 Class + Program.cs 调用）
- 文档更新完成，示例代码可运行
- 通过 AOT 编译测试
- 兼容 .NET Standard 2.0 和 .NET 10.0
