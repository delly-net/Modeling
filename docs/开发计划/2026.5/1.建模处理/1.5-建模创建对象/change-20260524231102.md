# 变更记录 - 1.5 建模创建对象

## 备份信息

**原始文档备份**: [backup-20260524231102.md](./backup-20260524231102.md)

## 变更内容

### 1. 新增接口方法

**IBaseModel 接口新增方法**：
```csharp
/// <summary>
/// 创建模型类型的新实例
/// </summary>
/// <returns>模型类型的新实例</returns>
object CreateInstance();
```

### 2. 源生成扩展

**ModelableSourceGenerator 新增生成内容**：
- 在生成的 `mc__{ClassName}` 模型类中实现 `CreateInstance()` 方法
- 方法返回 `new {ClassName}()` 创建的新实例

**ModelTableSourceGenerator 新增生成内容**：
- 在生成的 `emc__{ClassName}` 实体模型类中实现 `CreateInstance()` 方法
- 方法返回 `new {ClassName}()` 创建的新实例

### 3. 兼容性要求

- 同时兼容 `IModel` 与 `IEntityModel` 的源生成处理
- `IModel` 继承自 `IBaseModel`
- `IEntityModel` 继承自 `IBaseModel`

### 4. 设计约束（第一次返工新增）

**构造函数约束**：
- 所有使用 `Modelable` / `MoTable` / `MoQuery` 特性的对象，**不允许定义带参数的构造函数**
- 仅支持无参构造函数

**实现方式**：
```csharp
// ✅ 正确 - 无参构造函数
[Modelable]
public partial class User
{
    public User() { }
    public string Id { get; set; } = string.Empty;
}

// ❌ 错误 - 带参数的构造函数
[Modelable]
public partial class User(string id)  // 不允许
{
    public string Id { get; } = id;
}

// ✅ 正确 - 隐式无参构造函数
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
}
```

**Demo 编写规范**：
- Demo 的编写应该建立 `功能特有Class` 再由 `Program.cs` 调用
- 不要将所有代码都放到 `Program.cs` 中

## 变更原因

根据需求补充文档的反馈（第一次返工）：
1. 为了简化 `CreateInstance()` 的实现，避免处理复杂的构造函数参数
2. 为了提高代码可维护性，Demo 应该有良好的代码组织结构

## 可能产生的影响

1. **接口变更**：`IBaseModel` 接口新增方法，所有实现类需要实现该方法
2. **源生成器变更**：
   - `ModelableSourceGenerator` 需要添加 `CreateInstance()` 方法生成
   - `ModelTableSourceGenerator` 需要添加 `CreateInstance()` 方法生成
3. **现有模型类**：需要更新 `Models` 目录下的模型类实现 `CreateInstance()` 方法
4. **AOT 兼容性**：`CreateInstance()` 使用 `new T()` 语法，AOT 友好
5. **类型安全**：返回 `object` 类型，需要使用者进行类型转换
6. **用户代码约束**：使用建模特性的类不能使用带参数的构造函数

## 差异点总结

| 项目 | 原始设计 | 新设计 |
|:-----|:---------|:-------|
| 创建方式 | 静态方法 `User.Create()` | 模型方法 `model.CreateInstance()` |
| 调用入口 | `User.Create()` | `User.GetModel().CreateInstance()` |
| 返回类型 | `User`（强类型） | `object`（弱类型） |
| 接口位置 | 静态扩展 | IBaseModel 接口方法 |
| 源生成器 | 仅 ModelableSourceGenerator | ModelableSourceGenerator + ModelTableSourceGenerator |
| 模型类型 | IModel | IModel + IEntityModel |
| 构造函数支持 | 支持无参和主构造函数（含默认值） | 仅支持无参构造函数 |
| Demo 编写 | 可直接在 Program.cs 中 | 需要建立功能特有 Class |
