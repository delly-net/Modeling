# 1.5 建模创建对象

## 任务信息
- **任务号**: 1.5
- **任务名称**: 建模创建对象
- **责任人**: 郑琳
- **任务类型**: 功能开发
- **任务描述**: 添加建模直接创建对象实例的能力

## 功能概述

为 `[Modelable]` 类添加源生成能力，使其能够通过模型对象直接创建新实例。此功能允许用户通过 `IModel` 接口或模型单例以类型安全的方式创建对象实例，无需知道具体的类型或构造函数参数。

**重要约束**：
- 必须保持与现有功能的向后兼容性
- 生成的代码需要支持 AOT 编译
- 需要处理不同类型的构造函数（无参、主构造函数、传统构造函数）
- 对于只读属性，需要在创建时通过构造函数或初始化器设置

## 技术背景

### 现有实现

**Modelable 特性**：
```csharp
[AttributeUsage(AttributeTargets.Class, Inherited = false, AllowMultiple = false)]
public sealed class ModelableAttribute : Attribute
{
}
```

**现有使用方式**：
```csharp
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
    public string Name { get; set; } = string.Empty;
    public string? Password { get; set; }
}

// 通过静态方法获取模型
var model = User.GetModel();

// 通过模型访问属性
var properties = model.GetProperties();
var idProperty = model.GetProperty(nameof(User.Id));
```

**现有生成的模型结构**：
```csharp
// 生成的模型类
public class mc__User : IModel
{
    public static mc__User Instance => _instance;
    public string Name => "UserModel";
    public string Namespace => "...";
    public IModelProperty[] GetProperties() { ... }
    public IModelProperty? GetProperty(string name) { ... }
}

// User 类中的静态方法
public partial class User
{
    public static mc__User GetModel() => mc__User.Instance;
}
```

### 新增需求

为模型接口添加创建对象的能力：

```csharp
/// <summary>
/// 创建模型类型的新实例
/// </summary>
/// <returns>模型类型的新实例</returns>
public partial class User
{
    /// <summary>
    /// 通过模型创建新实例
    /// </summary>
    /// <param name="model">模型对象</param>
    /// <returns>新实例</returns>
    public static User Create(IModel model);

    /// <summary>
    /// 创建新实例
    /// </summary>
    /// <returns>新实例</returns>
    public static User Create();
}
```

### 现有接口

**IModel 接口**：
```csharp

```

## 实现方案

### 1. 具体步骤

1. **扩展 ModelableSourceGenerator**
   - 分析目标类的构造函数（主构造函数、传统构造函数、无参构造函数）
   - 为主构造函数生成对应的 `Create` 方法重载
   - 生成默认的 `Create()` 方法（无参）

2. **生成 Create 方法**
   - `Create()`: 使用无参构造函数或 `new T()` 创建实例
   - `Create(params object[] args)`: 根据参数类型匹配最合适的构造函数

3. **添加单元测试**
   - 测试无参 `Create()` 方法
   - 测试带参 `Create()` 方法
   - 测试主构造函数类的创建
   - 测试传统构造函数类的创建

4. **更新文档**
   - 更新用户指南，说明创建对象的方法

### 3. 关键实现细节

**主构造函数支持**：
```csharp
// 源代码
[Modelable]
public partial class User(string id)
{
    public string Id { get; } = id;
    public string Name { get; set; } = string.Empty;
}

// 生成的代码
public partial class User
{
    public static User Create() => new User(string.Empty);

    public static User Create(string id) => new User(id);
}
```

**传统构造函数支持**：
```csharp
// 源代码
[Modelable]
public partial class User
{
    public User() { }
    public User(string id) { Id = id; }

    public string Id { get; set; } = string.Empty;
}

// 生成的代码
public partial class User
{
    public static User Create() => new User();

    public static User Create(string id) => new User(id);
}
```

**仅无参构造函数**：
```csharp
// 源代码
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
}

// 生成的代码
public partial class User
{
    public static User Create() => new User();
}
```

## 使用示例

### 示例 1：基本使用

```csharp
[Modelable]
public partial class User
{
    public string Id { get; set; } = string.Empty;
    public string Name { get; set; } = string.Empty;
    public string? Password { get; set; }
}

// 使用 Create() 创建实例
var user1 = User.Create();
user1.Id = "1";
user1.Name = "张三";

var user2 = User.Create();
user2.Id = "2";
user2.Name = "李四";
```

### 示例 2：主构造函数

```csharp
[Modelable]
public partial class User(string id)
{
    public string Id { get; } = id;
    public string Name { get; set; } = string.Empty;
}

// 使用 Create(string) 创建实例
var user = User.Create("user-123");
user.Name = "张三";

// 使用无参 Create() 创建实例（使用默认值）
var defaultUser = User.Create();
// id 为 string.Empty
```

### 示例 3：传统构造函数

```csharp
[Modelable]
public partial class User
{
    public User() { }
    public User(string id) { Id = id; }

    public string Id { get; set; } = string.Empty;
    public string Name { get; set; } = string.Empty;
}

// 无参创建
var user1 = User.Create();

// 带参创建
var user2 = User.Create("user-456");
```

### 示例 4：多参数构造函数

```csharp
[Modelable]
public partial class Order(string id, long userId, decimal amount)
{
    public string Id { get; } = id;
    public long UserId { get; } = userId;
    public decimal Amount { get; } = amount;
    public DateTime CreatedAt { get; set; } = DateTime.UtcNow;
}

// 使用 Create() 创建默认实例
var defaultOrder = Order.Create();

// 使用 Create(...) 创建带参实例
var order = Order.Create("order-123", 1, 99.99m);
order.CreatedAt = DateTime.UtcNow;
```
