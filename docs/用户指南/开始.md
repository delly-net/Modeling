# 开始使用

## 快速入门

### 1. 安装

通过 NuGet 安装 Delly.Modeling 包：

```bash
dotnet add package Delly.Modeling
```

### 2. 基本用法

创建一个可建模的类：

```csharp
using Delly.Modeling;

[Modelable]
public partial class User(string id)
{
    public string Id { get; } = id;
    public string Name { get; set; } = string.Empty;
    public string? Password { get; set; }
}
```

### 3. 使用生成的模型

```csharp
// 获取模型实例
var model = Model<User>.Instance;

// 获取用户实例
var user = new User("user-123");

// 获取所有属性名
var properties = model.GetProperties();
// 返回: ["Id", "Name", "Password"]

// 通过名称获取属性值
var name = ModelUtils.GetProperty(model, user, "Name");

// 通过名称设置属性值
ModelUtils.SetProperty(model, user, "Name", "John Doe");
```

## 关键要求

- 类必须标记 `[Modelable]` 属性
- 类必须声明为 `partial`
- 支持构造函数和主构造函数语法

## 生成的功能

为每个 `[Modelable]` 类生成以下扩展方法：

- `GetProperties()` - 返回属性名称数组
- `GetProperty(model, instance, name)` - 按名称获取属性值
- `SetProperty(model, instance, name, value)` - 按名称设置属性值

## 建模集合（MoSet）

使用 `[MoSet]` 属性自动收集程序集中所有标记了 `[MoTable]` 或 `[MoQuery]` 的模型：

```csharp
using Delly.Modeling;

[MoSet]
public partial class EntityModelSet;
```

生成的代码实现 `IEntityModelSet` 接口，提供以下功能：

### 非泛型方法

```csharp
// 获取所有模型
var allModels = entitySet.GetModels();
var count = entitySet.Count;
```

### 泛型方法

```csharp
// 获取指定类型的模型（类型不匹配时抛出 NotSupportedException）
var userModel = entitySet.GetModel<UserEntity>();

// 尝试获取指定类型的模型（未找到时返回 null）
var user = entitySet.TryGetModel<UserEntity>();
if (user != null)
{
    Console.WriteLine(user.ClassName);
}
```

## 建模工厂（IEntityModelFactory）

使用建模工厂统一管理多个建模集合：

```csharp
using Delly.Modeling;

// 定义建模集合
[MoSet]
public partial class EntityModelSet;

// 定义模型类
[MoTable("users")]
public partial class User
{
    public long Id { get; set; }
    public string Name { get; set; }
}

[MoTable("orders")]
public partial class Order
{
    public long Id { get; set; }
    public long UserId { get; set; }
}

// 使用工厂
var factory = new DefaultEntityModelFactory();
var entitySet = new EntityModelSet();

// 注册集合（必需，否则泛型方法无法获取）
factory.AddSet(entitySet);

// 通过泛型获取模型
var userModel = factory.GetModel<User>();
Console.WriteLine($"表名: {userModel.Name}");

// 使用 TryGetModel 避免异常
var productModel = factory.TryGetModel<Product>();
if (productModel != null)
{
    Console.WriteLine($"表名: {productModel.Name}");
}

// 通过类全称获取
var userModel2 = factory.GetModel("MyApp.Models.User");

// 通过表名获取
var orderModel = factory.GetModelByTableName("orders");
```

**重要提示**：
- 泛型方法 `GetModel<T>()` 和 `TryGetModel<T>()` 只在通过 `AddSet` 方法注册的集合中查找
- 直接使用 `Add` 方法注册的模型无法通过泛型方法获取
- 同名的 `IEntityModelSet` 只会被注册一次，后续调用将被忽略
