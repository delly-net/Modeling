# Delly.Modeling

一个 .NET 建模库，通过源代码生成器提供对象建模能力。它支持在编译时进行类似反射的属性访问和操作。

## 代码文件索引

```
Delly.Modeling

│  ├─ ModelableAttribute.cs  # 标记类为可建模的特性
│  ├─ MoTableAttribute.cs    # 标记类为实体表的特性

│  ├─ MoSetAttribute.cs      # 标记类为建模集合的特性
│  ├─ MoColumnAttribute.cs   # 标记实体列的特性
│  ├─ MoColumnIndexAttribute.cs  # 标记列索引的特性
│  └─ ParsableAttribute.cs   # 标记自定义 Parser 类的特性
├─ Interfaces/
│  ├─ IBaseModel.cs          # 基础模型接口，定义核心模型功能

│  ├─ IEntityModel.cs        # 实体模型接口，定义表/查询对象模型

│  ├─ IEntityModelSet.cs    # 实体模型集合接口，自动收集实体模型
```
